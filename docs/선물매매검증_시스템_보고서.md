# 선물매매검증 시스템 상세 보고서
> 이 문서는 선물매매검증 시스템의 전체 구조와 운영 방식을 설명합니다.
> 최종 업데이트: 2026-04-22

## 시스템 파일 목록
- lib/kis-client.ts — KIS API 클라이언트 (토큰/근월물코드/분봉수집)
- lib/futures-market-data.ts — 시장 데이터 수집 (Yahoo+KIS)
- lib/futures-trading-store.ts — Redis 저장/로드 함수
- lib/futures-claude-analyzer.ts — 클로드 분석 함수 모음
- lib/types/futures-trading.ts — 타입 정의
- app/futures-trading/page.tsx — UI
- app/api/futures-trading/route.ts — 매매기록 API
- app/api/futures-trading/reply/route.ts — 답변 API
- app/api/futures-trading/pattern/route.ts — 패턴 API
- app/api/futures-trading/quantified/route.ts — 수치화 API
- app/api/futures-trading/dynamic-symbols/route.ts — 동적심볼 API
- app/api/cron/collect-market-data/route.ts — 일일 시장데이터 수집 Cron

## Redis 키 구조
- futures-trading:records — 매매 기록 전체 (qaThreads 포함)
- futures-trading:quantified — 수치화 완료/불가 조건 목록
- futures-trading:pattern — 클로드가 자동 업데이트하는 매매 패턴
- futures-trading:market-data:{date} — 날짜별 시장 분봉 데이터
- futures-trading:market-data-index — 데이터 보유 날짜 목록
- futures-trading:dynamic-symbols — 메모에서 자동 감지된 추가 수집 심볼
- futures-trading:kis-token — KIS API 토큰 캐시 (24시간 유효)
- futures-trading:kis-futures-code — 코스피200 선물 근월물 코드 캐시
- futures-trading:messages — 자유전달판 메시지

## 매매 기록 저장 시 자동 실행 흐름
1. Redis에 기록 저장
2. 메모에서 새 지표 자동 감지 (화이트리스트 11개 심볼)
3. 해당 날짜 시장 데이터 수집 (Yahoo + KIS API)
4. 클로드가 데이터 보며 패턴 분석 → qaThreads/quantified 자동 추가
5. 전체 매매 패턴 자동 업데이트

## 수집 심볼
- 005930.KS (삼성전자) — Yahoo Finance
- 000660.KS (SK하이닉스) — Yahoo Finance
- KOSP200F (코스피200 선물 근월물) — KIS API, 자동 롤오버
- ES=F (S&P500 선물) — Yahoo Finance
- DX-Y.NYB (달러인덱스) — Yahoo Finance
- 동적 심볼 — 메모/댓글에서 자동 감지

## Cron 스케줄
- KST 16:30 (UTC 07:30), 평일 자동 실행
- 데이터 품질 검증 후 저장
